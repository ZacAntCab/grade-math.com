Grades
Grade must be between 0 and 100. Please input a grade or remove the input box. Please input the percentage as a whole number between 0 and 100. Percentage inputs do not add up to 100.
This system is quite simple and as long as you know what percentage of your final grade each grade is, the accuracy will not vary depending on the schools grading system, unlike the other calculators.
